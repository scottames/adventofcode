# Advent of Code

@scottames' advent of code solutions - just a bit of fun all in the hopes to have fun, explore & improve

## Usage

### Golang

```shell
mage go:run <year> <day>
```

### Alternatively

```shell
mage -l
```

## Requirements

1. [Go](https://golang.org/)
1. [Mage](https://magefile.org/)
1. [Rust](https://www.rust-lang.org/)
